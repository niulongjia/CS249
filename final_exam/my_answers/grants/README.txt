my_answers/grants/ folder contains following files:

- Solution_GrantApplications.R (the R script for Grant Applications)
- grantData.RData (data to be loaded)
- APM_prediction_input.csv (test set)
- predictions.csv (predictions for the test set)
- README.txt

Results in predictions.csv contain two columns, the first column is 
the probability for 1 (successful), the second column is the probability 
for 0 (unsuccessful).
